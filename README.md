# Trcobao
Sistema para administrar las actividades que un profesor le deja a sus alumnos

Un profesor puede agregar grupos y actividades, se pretende que cada grupo tenga una contrase√±a la cual el profesor debe compartir a sus alumnos para que estos puedan integrarse.

## Requisitos
  1. Docker
  2. Docker-compose

## Probar el proyecto en modo desarrollo
  1. Clonar el proyecto
   ```bash 
   git clone https://github.com/CronoxD/trcobao.git
   ```
  2. Ejecutar docker-compose
  ```
  cd trcoabo
  docker-compose up --build
  ```
